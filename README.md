# app-paises

Um jogo interativo desenvolvido em **Node.js** onde o usuário deve adivinhar o nome do país através de sua bandeira. O projeto conta com sistema de autenticação, persistência de dados em JSON e contagem de experiência (XP).

Link : https://app-paises-4956.onrender.com

## Tecnologias Utilizadas

* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
* **Backend:** Node.js, Express.js
* **Banco de Dados:** Arquivo JSON (Persistência local)
* **Deploy:** Render
